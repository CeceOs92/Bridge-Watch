# Analytics

## Overview
The Analytics dashboard provides deep insights into bridge volume, asset liquidity, and historical health metrics.

## Key Features
1. **Volume Analysis**: View daily, weekly, or monthly transfer volumes across all supported bridges.
2. **Liquidity Depth**: Monitor liquidity pool sizes on interconnected DEXs.
3. **Health Trends**: Track how asset health scores have evolved over time.

## Custom Charts
You can customize the timeframes and overlay multiple assets on the same chart for comparative analysis. Click **Save View** to keep your custom chart configuration for future visits.
